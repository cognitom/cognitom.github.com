No more spaghetti HTML generators.
No more odd tags on preview.

No hassle programing needed. HTML is generated by HTML files.

milk.coffee はクライアントサイドで動作するテンプレートフレームワークです。
ドキュメントとテンプレート、そしてAjaxなどで取得したデータを合わせ、一枚のページをレンダリングします。
JavaScriptによって全て処理されるため、サーバサイドのプログラムは基本的に不要です。

No more mockups needed. You can design on your template.

モックアップの作成と、テンプレートの作成は別の話、と思っていませんか? 確かに多くのテンプレートエンジンではテンプレート≒HTMLですよね。WEBブラウザでテンプレートを表示すると、変な記号が表示されたり、複数並んでないとイメージが掴めないのに、1つしか表示されなかったり…。milk.coffeeならそんなことはありません。
milkのテンプレートは完全なHTMLなので、ビジュアルエディタ(Dreamweaver)での編集もOKです。

No database needed. Just put your docs to folder.

フォルダ内のファイル検索は、JavaScriptからできないことの一つでした。milkはHTMLファイル検索に特化した、SQLライクなクエリ言語を提供します。

例えば、カレントディレクトリの最新3件のページ情報が必要なら...
SELECT * FROM ./ LIMIT3
のように指定するだけで、テンプレートから情報にアクセスできます。

※hSQL = html Search Query Language
